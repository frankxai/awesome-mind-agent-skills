# Awesome Mind Agent Skills

**Curated discovery layer for open-source projects, tools, papers, MCP servers, skills, and frameworks relevant to mind intelligence, second brains, research agents, neuroscience, psychology, and learning.**

## Purpose
The ecosystem map that helps humans and agents discover high-quality building blocks. Complements the core swarm repos with external excellence.

## How People Experience It
Browse categories to find tools to integrate into your vault or research packs. Contributors add verified entries.

**Experience**: Start with README categories → drill into specific category.md → evaluate for your use case (personal OS or research).

## How Agents Explore It
Agents parse categories/*.md for relevant tools, then propose integrations via skills or workflows in the main repos.

**Example Prompt**:
"Scan awesome-mind-agent-skills categories for tools supporting BIDS + structured outputs. Recommend 3 that integrate with neuroscience-research-intelligence-system using latest MNE/NWB standards."
